Лабораторная работа №4 по дисциплине «Интеллектуальный анализ данных».
Реализация алгоритмов Apriori и FP‑Growth «с нуля» для поиска ассоциативных правил, эксперименты на реальном и синтетических наборах данных, визуализация и интерпретация результатов.

📌 Цель работы

Освоить методы поиска ассоциативных правил, реализовать алгоритмы Apriori и FP‑Growth без использования готовых библиотек, применить их к транзакционным данным, провести сравнительный анализ производительности и содержательно интерпретировать найденные правила.
